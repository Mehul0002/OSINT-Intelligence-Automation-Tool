# OSINT-Intelligence-Automation-Tool
The OSINT Intelligence Automation Tool is a Python-based GUI application designed to automate the collection and analysis of open-source intelligence (OSINT) data. It helps security researchers, students, and analysts gather publicly available information in a legal, ethical, and structured manner.
